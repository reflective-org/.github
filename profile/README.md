## Welcome to Reflective's GitHub! 

Reflective is a non-profit research organization aiming to radically accelerate the pace of sunlight reflection research. We equip the world with the data and tools needed to make informed decisions about sunlight reflection, fast enough to matter.


Learn more at [reflective.org](reflective.org).

***

At Reflective, we believe that decisions shoud not be made in the dark and that means that all of our code is open source. 
> [!NOTE]
> We are still in the process of open sourcing all of our repositories. This note being removed from this README will signify that we have completed that process. 

***
### Multi-Scale Modeling Related Repos
- [MAM4-Jax](https://github.com/reflective-org/MAM4-JAX) is a JAX port of the MAM4 aerosol-microphysics box model, validated against the E3SM Fortran reference.
- [carma-jax](https://github.com/reflective-org/carma-jax) is a JAX port of CARMA (Community Aerosol and Radiation Model for Atmospheres).
- [tomas-jax](https://github.com/reflective-org/tomas-jax) is a JAX port of the TOMAS (TwO-Moment Aerosol Sectional) aerosol microphysics model.

***

### SAI Simulator Related Repos
- [sai-simulator](https://github.com/reflective-org/sai-simulator) is a tool to explore the effects of stratospheric aerosol injection on the climate.

### Reflective Cloud Hub
The Reflective Cloud Hub is a community hub for open-source and reproducible SRM research. This is a free resource available to all SRM researchers.
Everything pertaining to the Reflective Cloud Hub lives in a seperate GitHub organization, [ReflectiveCloud](https://github.com/ReflectiveCloud).
